# Store listing

## App name

Noren VPN

## Short description

One-tap VPN for Android TV using public servers, with optional Pro controls.

## Full description

Noren VPN is a simple VPN utility for Android TV that connects to public volunteer servers listed by the VPN Gate Academic Experiment Project.

Start a VPN connection with the remote, see the current connection status, and check the public IP after connecting. Automatic mode selects a candidate server so the app stays easy to use from the couch.

For users who want more control, Noren VPN Pro unlocks manual server selection, server sorting, quality filtering, manual switching, and OVPN profile export through a one-time Google Play purchase.

Key features:

- One-click VPN connection for Android TV
- Remote-friendly TV interface
- Automatic server candidate selection
- Public IP check after connecting
- Optional Pro server selection and sorting
- Optional OVPN profile export for compatible OpenVPN apps

Important notes:

- VPN connections use independent public VPN Gate volunteer servers. The app developer does not operate these servers and is not affiliated with VPN Gate.
- VPN Gate states that connection logs include source IP/hostname, destination host/IP/port metadata, traffic totals, and errors. Central connection logs are kept for at least three months, and volunteer servers keep TCP/IP packet headers for at least two weeks.
- Server availability, quality, speed, and location can change at any time.
- The app does not guarantee anonymity, security, streaming access, or access to any specific website or service.
- Pro sells app control features only; it does not provide private or premium VPN servers.
- Corresponding source code and open-source notices: https://github.com/sclee980528-prog/noren-vpn-source

## Japanese listing (ja-JP)

### App name

Noren VPN

### Short description

公開ボランティアサーバーへワンタップ接続。Android TV対応、Proで詳細操作。

### Full description

Noren VPNは、VPN Gate学術実験プロジェクトが公開する独立ボランティアサーバーへ接続する、Android TV向けのシンプルなVPNツールです。

リモコンでVPN接続を開始し、現在の接続状態や接続後の公開IPを確認できます。自動モードが接続候補を選ぶため、テレビから少ない操作で利用できます。

より細かく操作したい場合は、Google Playの買い切り型Proで、サーバーの手動選択、並べ替え、品質フィルター、手動切替、OVPNプロファイルの書き出しを利用できます。

主な機能:

- Android TVでのワンタップVPN接続
- リモコン操作に適した画面
- 接続候補の自動選択
- 接続後の公開IP確認
- Proによるサーバー選択と並べ替え
- 対応OpenVPNアプリ向けOVPNプロファイル書き出し

重要事項:

- 接続には独立した公開VPN Gateボランティアサーバーを使用します。アプリ開発者はこれらのサーバーを運営しておらず、VPN Gateとの提携関係もありません。
- VPN Gateによると、接続元IP・ホスト名、接続先ホスト・IP・ポートのメタデータ、通信量、エラーが記録されます。中央接続ログは3か月以上、ボランティアサーバーのTCP/IPパケットヘッダーは2週間以上保存されます。
- サーバーの稼働状況、品質、速度、所在地は随時変わります。
- 匿名性、安全性、動画配信サービス、特定サイトやサービスへの接続は保証しません。
- Proが提供するのはアプリの操作機能だけです。専用またはプレミアムVPNサーバーは提供しません。
- 対応するソースコードとオープンソースライセンス表示: https://github.com/sclee980528-prog/noren-vpn-source
